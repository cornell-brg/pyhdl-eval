
Implement a hardware module named TopModule with the following interface.
All input and output ports are one bit unless otherwise specified.

 - parameter nports
 - parameter nbits
 - input  in_ (array of nports, each port is nbits)
 - input  sel (log2(nports))
 - output out (nbits)

The module should implement a M-to-1 multiplexer that is parameterized by
the number of input ports (i.e., M or nports) and the bitwidth of the
input and output ports (nbits). The parameter nports can be assumed to be
a power of two. The input sel should select which of the inputs is used
to write the output. If sel equals 0 then in_[0] should be selected, if
sel equals 1 then then in_[1] should be selected, and so on.
